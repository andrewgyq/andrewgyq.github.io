Java中文编码

2015-05-17

概述

计算机中存储的最小单位是一个字节，8bit(位)即1byte(字节)，所能表示的字符范围为0-255。但需要表示的符号太多，无法用一个字节来完全表示，需要定义char数据结构，char到byte必须编码。

Java中系统的默认编码方式为GBK，中文占2个字节，而英文字符占1个字节。

char[] => byte[]

String str = "I am 君山";
byte[] iso8859 = str.getBytes("ISO-8859-1");
for (byte b : iso8859) {
    System.out.print(String.format("%02x ", b));
}
System.out.println();

byte[] gbk = str.getBytes("GBK");
for (byte b : gbk) {
    System.out.print(String.format("%02x ", b));
}
System.out.println();


byte[] utf8 = str.getBytes("UTF-8");
for (byte b : utf8) {
    System.out.print(String.format("%02x ", b));
}
System.out.println();

ISO-8859-1编码

 I        a  m      君 山

49 20 61 6d 20 3f 3f

ISO-8859-1是单字节编码，中文"君山"被转化成3f (?)，出现了中文乱码，大部分基础的Java框架或系统的默认字符集编码是ISO-8859-1

GBK编码

 I        a  m        君     山

49 20 61 6d 20 be fd c9 bd

GBK编码中，中文占2个字节，而英文字符占1个字节

UTF-8编码

 I        a  m           君          山

49 20 61 6d 20 e5 90 9b e5 b1 b1

UTF-8编码中，对单字节范围内字符仍然用一个字节表示，对汉字采用三个字节表示。
