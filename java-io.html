Java IO类库基本架构

Java的IO操作类在java.io下，有近80个类，分为以下四组：

  • 基于字节IO操作接口：Inputstream和OutputStream
  • 基于字符IO操作接口：Writer和Reader
  • 基于磁盘IO操作接口：File
  • 基于网络IO操作接口：Socket

基于字节IO操作接口

操作数据的方式可以组合使用

OutputStream out = new BufferedOutputStream(new ObjectOutputStream(new FileOutputStream("fileName")));
