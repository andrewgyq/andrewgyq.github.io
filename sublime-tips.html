Sublime Text使用技巧

2015-11-17

显示空格符

// Preferences.sublime-settings
"draw_white_space": "all"

空格缩进

// Preferences.sublime-settings
"translate_tabs_to_spaces": true

删除行

{ "keys": ["ctrl+d"], "command": "run_macro_file", "args": {"file": "Packages/Default/Delete Line.sublime-macro"}}

分屏

Alt + Shift + 2

搜索文件

Ctrl + P

打开命令面板

Ctrl + Shift + P
