Ubuntu编译安装Python3.4.3

2015-06-03

Ubuntu14.04中默认安装Python2.7.6在/usr/bin/python2.7，现编译安装Python3.4.3，实现共存互不影响

1.安装依赖

sudo apt-get install build-essential libssl-dev libevent-dev libjpeg-dev libxml2-dev libxslt-dev

2.下载安装包解压

wget https://www.python.org/ftp/python/3.4.3/Python-3.4.3.tgz
tar xf Python-3.4.3.tgz
rm Python-3.4.3.tgz

3.安装

./configure --prefix=/home/ubuntu
make && make altinstall

Python3.4.3装在/home/ubuntu/bin/python3.4

4.配置环境变量

vi .bashrc  
export PATH=$PATH:/home/ubuntu/bin  
source .bashrc

5.安装virtualenv

pip2.7 install virtualenv

virtualenv装在/home/ubuntu/lib/python3.4/site-packages

6.创建项目使用Python3.4.3环境

mkdir app
cd app  
virtualenv-3.4 buildenv  
source buildenv/bin/activate

输入python，将会出现Python3.4.3

在Python3.4.3环境下安装包，pip install requests，将装在/home/ubuntu/app/buildenv/lib/python3.4/site-packages，为此项目所有

退出Python3.4.3，输入deactivate
