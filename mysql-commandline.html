MySQL指令大全

2015-06-01

1.root登录mysql

mysql -u root -p

2.修改root密码

UPDATE mysql.user SET Password=PASSWORD('root') WHERE User='root';
FLUSH PRIVILEGES;

3.启动mysqld服务

sudo service mysqld start

4.显示数据库

SHOW DATABASES;

5.创建数据库

CREATE DATABASE sakila;

6.使用数据库

USE sakila

7.删除数据库

DROP DATABASE sakila;

8.运行sql

mysql -h 127.0.0.1 -u root -p  --default_character_set utf8 sakila < *.sql

9.导出sql

mysqldump -u root -p sakila > sakila.sql

10.删除table

TRUNCATE TABLE test;
