Git指令大全

2015-06-04

1.设置

git config --global user.name "andrewgyq"
git config --global user.email "514362283@qq.com"

2.查看设置

git config -l

3.克隆项目

git clone https://github.com/andrewgyq/andrewgyq.github.io.git

4.查看状态

git status

5.查看历史记录

git log
git log --pretty=oneline

6.回退到上一个版本

git reset --hard HEAD^

7.查看每一次命令

git reflog

8.丢弃工作区的修改

git checkout -- file

9.添加到了暂存区想丢弃修改

git reset HEAD file
git checkout -- file

10.创建dev分支并切换

git checkout -b dev

11.查看当前分支

git branch

12.切换回master分支

git checkout master

13.把dev分支的工作成果合并到master分支上

git merge dev

14.删除dev分支

git branch -d dev
