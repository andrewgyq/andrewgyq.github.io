Confluence

2016-07-09

Confluence安装

1.下载atlassian-confluence-5.8.8.zip

2.解压文件准备绿色安装

3.配置环境变量，jdk要安装1.8版本，且安装目录名没有空格

JAVA_HOME = C:\Java\jdk1.8.0_91
JRE_HOME = C:\Java\jre8

4.编辑.\bin\setenv.bat，注释掉含有-Xloggc的行

5.编辑.\confluence\WEB-INF\classes\confluence-init.properties

confluence.home=d:/confluence/data

6.启动.\bin\start-confluence.bat

7.浏览器输入http://localhost:8090

Confluence Linux Install

1.下载atlassian-confluence-5.10.4-x64.bin

2.添加可执行权限

chmod +x atlassian-confluence-5.10.4-x64.bin

3.执行安装

./atlassian-confluence-5.10.4-x64.bin

Installation Directory: /opt/atlassian/confluence
Home Directory: /var/atlassian/application-data/confluence
HTTP Port: 8090
RMI Port: 8000
Install as service: Yes

4.关闭应用

cd /opt/atlassian/confluence/bin
./stop-confluence.sh

5.添加mysql驱动

cp mysql-connector-java-5.1.15.jar /opt/atlassian/confluence/confluence/WEB-INF/lib
chmod 644 /opt/atlassian/confluence/confluence/WEB-INF/lib/mysql-connector-java-5.1.15.jar

6.启动应用

cd /opt/atlassian/confluence/bin
./start-confluence.sh

7.查看日志

cat /opt/atlassian/confluence/logs/catalina.out

8.卸载confluence

./uninstall
rm -rf /opt/atlassian/
rm -rf /var/atlassian/

MySQL集成

mysql>create database confluence character set UTF8;  
mysql>grant all on confluence.* to confluence@"%" identified by "confluence"; 
mysql>grant all on confluence.* to confluence@"localhost" identified by "confluence";
