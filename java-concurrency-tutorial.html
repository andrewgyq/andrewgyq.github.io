Java并发

2016-05-04

1.Java并发和多线程教程

过去的时候计算机只含有单个CPU，同一时刻只能执行单一程序。以后出现了多任务，计算机可以同时执行多个任务。尽管不是真正意义上的同时执行。而单个CPU被程序之间共享，操作系统可以在程序运行期间切换，在切换之前执行一会程序。

多任务的出现也给软件开发者带来了新的挑战。程序不再能拥有所有的CPU时间、内存和其它计算资源。一个好的程序应该释放所有的资源当它不再使用以使得其它程序使用。
