Linux指令大全

2015-06-02

脚本添加执行权限

chmod +x start.sh

从该机器拷贝文件夹到远程主机

scp -r folder root@192.168.0.1:/home/andrewgyq/test

从远程主机拷贝文件夹到该机器

scp -r root@192.168.0.1:/home/andrewgyq/test /home/andrewgyq

开端口远程访问

sudo vi /etc/sysconfig/iptables
-A INPUT -p tcp -m tcp --dport 8000 -j ACCEPT
-A INPUT -p tcp -m tcp --dport 8080 -j ACCEPT
sudo service iptables restart

修改DNS

echo "nameserver 192.168.128.37" >> /etc/resolv.conf

关闭端口

sudo netstat -ap | grep :8080
kill -9 16158

查看服务是否开启

ps -ef | grep mysqld

查看安装包

sudo dpkg -l

删除安装包

sudo apt-get purge sougoupinyin
sudo apt-get autoremove

压缩文件夹

tar czf folder.tar.gz folder

建立一个不同的链接

ln -s /opt/ibm/java-ppc64-71 java

后台定时任务

crontab –e
#每隔一分钟执行一次
* * * * * /home/andrewgyq/test_cron.sh
#test_cron.sh
#!/bin/sh      
date >> time.txt
